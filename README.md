## 프로젝트 요약 
직원 퇴사의 주요 요인을 파악하고 개선점을 제시하기 위해 Kaggle IBM HR 데이터를 활용하여 퇴사자 예측 모델을 개발한 프로젝트입니다


## ✅진행 배경 
- **목표 및 방향성**: 퇴사 요인을 사전에 파악해 퇴사율 감소를 목표로 하였음


## 📊성과 및 결과  
- **결과물**:퇴사자 예측 모델 개발 (정확도: 98.64%), SHAP 분석으로 퇴사에 영향을 미치는 주요 요인(업무 만족도, 승진 기회 등) 파악 및 개선 전략 제안
- **성과**: 데이터 기반의 퇴사 방지 전략 제시

 
## 🛠사용 언어와 라이브러리 
- **프로그래밍 언어**: Python 
- **라이브러리 및 툴**: TensorFlow, NumPy, pandas, SHAP 등


## 📌프로젝트 진행 과정 
- **작업 단계**: 데이터 전처리 -> 모델 선정 및 구축 -> 파생변수 탐색 -> SHAP 분석 -> 개선 전략 제시
- **역할 분담**: 데이터 전처리, 모델 설계, 파생변수 탐색, ppt 제작 등


## ❗️한계점 및 어려웠던 점 
- **문제**: 모델 구축 중 데이터 불균형, 과적합 등의 문제
- **해결 방법**: SMOTE, 정규화(배치, L2, 얼리스탑핑, 드롭아웃 적용
- **아쉬운점**: kaggle에 올려 등수 확인 불가, 실제 회사 데이터를 적용하지 못해봄


## 💻학습 및 성장 
- **기술 향상**: 신경망 모델 최적화 및 SHAP 분석 역량 강화
- **팀워크 성장**: 의견을 편하게 나눌 수 있는 분위기를 조성하는 것의 중요성을 알게되었습니다. 
- **개인적 성장**: 논문 연구를 통해 데이터 근거 기반의 프로젝트 수행 능력 향상
